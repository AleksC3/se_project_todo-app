# Simple Todo App

A clean, responsive to-do list application built with vanilla JavaScript using ES6 classes and modular architecture. Users can create, complete, and delete tasks while managing optional due dates through an interactive popup form.

## Functionality

The Simple Todo App allows users to:

View a list of preloaded to-do items rendered dynamically with JavaScript

Open and close a popup form to create new tasks

Create new to-do items using the Enter key or the Create button

Assign optional due dates to tasks

Mark tasks as completed or uncompleted

Delete tasks individually

Automatically generate unique IDs for each task using the uuid package

Close the popup by pressing the Escape key

Experience real-time form validation with dynamic error messages and disabled submit button logic

The project follows an object-oriented programming (OOP) approach, with each feature encapsulated in its own ES6 class.
